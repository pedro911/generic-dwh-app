Generic DWH Tool

This tool is being developed for testing data warehouse query performance. The goal is to compare performance using 3 different models:

  • Generic DWH
  • Star Schema
  • Snowflake Schema

The App is not yet fully implemented, planing to finish until Oct/2019

Data Source

Data come from TPC-H Decision Support Benchmark. More info at http://www.tpc.org/tpch/

Methodology

The TPC-H DB was first generated using HammerDB script with different sizes. A PHP ETL application was used to create the Generic DWH DB. Pentaho BI was used to ETL TPC-DB to Star and Snowflake schemas. And this application was developed to query data on those models.

The Generic DWH has 3 different DBs running:

  • Dynamic: the algorithm checks if there are already saved facts for the selected ratios-dimensions. If there are no facts saved yet, the system will create new reference objects (if needed), and insert new facts.
  • No Combinations: static DB where only standard generic DWH queries are executed.
  • All Combinations: static DB where all possible ratio-dimension combinations are already stored, the query is always the fastest one.
